本次更新主要优化了亮色模式下的界面配色、优化了移动端访问体验、重构移动端的后台表格显示以及文章编辑器显示效果，并修复了访问报告因时区问题导致变化率计算错误。

此外，我们优化了文章编辑器的性能，使其在长文编辑时更加流畅；还为设置界面增加了重置功能，允许恢复某项设置到初始值。感谢社区开发者的贡献，文档也更加详细了。

### 注意事项

- 本次版本更新可能无法使用基于 Github Fork 的自动更新功能。请参照 [文档](https://neutralpress.net/docs/feature/upgrade) 中的手动更新指南进行更新。Docker 部署方式的用户请直接使用自动更新功能，无需担心此问题。
- 为防止覆盖你的自定义配色方案，亮色模式的配色优化不会自动生效。请更新后前往设置页面，重置 `site.color` 配置项并保存，以应用新的配色方案。
